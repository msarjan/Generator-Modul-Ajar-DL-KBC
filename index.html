⚠️ Disclaimer & Ketentuan Penggunaan

  • Hasil AI bukan produk final. Modul ajar yang digenerate adalah draf awal berbasis AI. Guru wajib memeriksa, menyesuaikan, dan memvalidasi seluruh isi sebelum digunakan dalam pembelajaran.
  • Tanggung jawab konten. Pemilik aplikasi tidak bertanggung jawab atas kesalahan konten, ketidaksesuaian kurikulum, atau dampak dari penggunaan modul tanpa proses verifikasi oleh guru.
  • Hak cipta bahan ajar. Bahan ajar (PDF) yang Anda upload adalah tanggung jawab pengguna sepenuhnya. Pastikan Anda memiliki hak penggunaan atas file tersebut. Aplikasi tidak menyimpan atau mendistribusikan file Anda.
  • Privasi data. Data form dan file PDF Anda diproses langsung di browser dan dikirim ke API AI hanya untuk keperluan generate modul. Tidak ada data yang disimpan di server aplikasi ini.
  • Non-komersial. Aplikasi ini dikembangkan secara mandiri, bersifat non-komersial, dan tidak berafiliasi resmi dengan Kementerian Agama RI. Referensi kurikulum (KMA 1503/2025, Permendikdasmen No. 10/2025) digunakan semata untuk keperluan pedagogis.
